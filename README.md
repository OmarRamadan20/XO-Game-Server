XO Game Server is a Java-based server application responsible for managing all online game operations for the XO Game system.  
It handles client connections, user authentication, online invitations, game state management, player status tracking, and game result processing.

The server supports multiple clients simultaneously using multithreading and provides real-time communication between players.  
This project was developed as part of a **team final project**, with a focus on networking, concurrency, and clean server-side architecture.
